# Foggel
A hobby project to register birds seen in different locations.
The project is in progress and therefore not fully functional yet.

It is built with React/node.js using RestAPI and SQL Server.

The goal is to create an app that lets you:
- add new birds with pictures,
- register new birds for specified location, date and time,
- add new locations,
- show registered birds with filtering such as month and species.
